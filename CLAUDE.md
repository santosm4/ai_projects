## Workspace Overview

This is a personal AI projects workspace. It is currently in early stages — projects are organized as subdirectories under `Workspace/` and `HelloWorld/`.

## Structure

- `HelloWorld/` — scratch/test scripts
- `Workspace/` — active project directories (each project gets its own subdirectory)

## Notes

- No build system, package manager, or test runner is shared across projects. Each project directory may introduce its own.
- When starting a new project here, create a dedicated subdirectory under `Workspace/` and set up the appropriate tooling within it.
